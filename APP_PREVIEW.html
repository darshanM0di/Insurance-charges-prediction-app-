📋 About This Application

This application uses a machine learning model trained on insurance data to predict insurance charges based on personal health and demographic information. The model was trained using PyCaret, an AutoML framework.

🎯 Features

Online Mode: Enter individual patient information and get instant prediction
Batch Mode: Upload a CSV file with multiple records for bulk predictions

📊 Input Parameters

Age: 18-100 years
Sex: Male or Female
BMI: Body Mass Index (10-50)
Children: Number of dependents (0-10)
Smoker: Current smoking status
Region: Geographic region (SW, SE, NW, NE)

💡 How It Works

1. Enter patient information in the form
2. Click the "Predict" button
3. The ML model processes the input
4. Predicted insurance charge is displayed

🔒 Security

All predictions are performed locally. No data is stored or transmitted to external servers.
